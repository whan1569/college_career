# 다양한 사회 경험을 통한 성장과 통찰

저는 다양한 사회 경험을 통해 인간관계의 복잡성과 조직 운영의 현실을 깊이 이해하게 되었습니다.
군 복무에서 강한 조직 문화와 상하관계의 특성을 배웠고, 대학교 산학 실습과 포럼 참여를 통해 실무적 경험을 쌓았습니다.
또한 고등학교 재능기부, 가게 운영, LG헬로비전 DX DATA 교육 과정을 통해 기술, 소통, 조직 운영, 실리적 판단의 중요성을 체득했습니다.

## 해병대 복무 (2016.03 ~ 2017.12): 강한 조직 사회의 이해

* 해병대 복무 기간 동안 강한 상하관계 속에서 조직이 어떻게 운영되는지 체험했습니다.
* 군대는 명확한 지휘 체계를 중심으로 움직이기 때문에 상하관계의 장점과 단점을 자연스럽게 익힐 수 있었습니다.
* 명령이 명확하게 전달되고 신속한 실행이 이루어지는 장점이 있었지만, 수직적인 구조에서 오는 의사소통의 경직성도 경험했습니다.
* 이 경험을 통해 어떤 조직이든 상하관계의 균형을 유지하는 것이 중요하다는 것을 깨달았습니다.

## 대학교 산학회사 실습 (2021.03 ~ 2021.09): 실무 프로젝트를 통한 역량 검증

관련 자료: [산학실습 폴더](https://github.com/whan1569/college_career/tree/main/%EC%82%B0%ED%95%99%EC%8B%A4%EC%8A%B5)
관련 프로젝트: [TOPIK Repository](https://github.com/whan1569/TOPIK)

* 이론과 실무를 연결하는 중요한 기회로, 리스크 없이 나의 역량을 증명할 수 있는 환경에서 일했습니다.
* Android와 OpenCV를 연동한 차량 번호판 인식 프로젝트를 수행했습니다.
* YOLO 기반 번호판 영역 검출, Grayscale 변환, 노이즈 제거, 이진화, 영역 분할, 숫자 판별로 이어지는 이미지 처리 파이프라인을 경험했습니다.
* TOPIK 모의시험 및 학습 방법 추천 시스템을 기획하고 Node.js 기반 프로토타입을 구현했습니다.
* TOPIK 문제 번호별 고정 출제 유형을 활용하여 Cold Start 상황에서도 동작 가능한 규칙 기반 추천 구조를 설계했습니다.
* 실제로 모의시험을 응시하고, 틀린 문제 유형을 파악한 뒤 자주 틀리는 유형과 유사한 문제를 노출하는 핵심 추천 흐름을 구현했습니다.
* 프로젝트 전체 완성도에는 한계가 있었지만, 문제 정의와 추천 구조 아이디어를 긍정적으로 평가받아 관련 포럼 참석으로 이어졌습니다.

## 가게 운영 (2022.07 ~ 2023.06): 사람을 이해하고, 가치를 높이는 방식

* 고객과의 상호작용을 통해 다양한 요구와 기대를 관리하는 방법을 배웠습니다.
* 실리를 추구하는 경영의 중요성을 깨닫고, 완벽한 운영은 자본 조직에서 사실상 불가능함을 체감했습니다.
* 불완전한 현실 속에서도 최대한 효율적으로 운영하며 성과를 내는 법을 배운 소중한 경험이었습니다.
* 지출을 최소화하면서 매출을 극대화하고, 운영이 최상의 상태일 때 사업을 매각한 경험은 단순한 운영을 넘어 가치 향상의 본질을 체득한 사례였습니다.
* 단기적인 수익이 아니라 가치를 키워 파는 전략적 사고를 통해, 경영이 단순한 유지가 아닌 자산화의 과정임을 직접 확인했습니다.

## LG헬로비전 DX DATA 교육 이수 (2024.08 ~ 2025.02): 기술과 리더십의 결합

관련 자료: [LG헬로비젼DATA스쿨 폴더](https://github.com/whan1569/college_career/tree/main/LG%ED%97%AC%EB%A1%9C%EB%B9%84%EC%A0%BCDATA%EC%8A%A4%EC%BF%A8)
실제 구현 레포지토리: [team3_web Repository](https://github.com/whan1569/team3_web)
기획·분석 레포지토리: [Team3 Repository](https://github.com/whan1569/Team3)

* LG헬로비전 DX DATA 전문 교육 과정을 이수하며 AI, DevOps, 데이터 시각화, 데이터 처리 등 디지털 전환의 핵심 기술을 학습했습니다.
* 단순한 이론이 아니라, 실습과 과제 중심의 교육을 통해 현업 적용 가능성을 염두에 둔 실전 역량을 키웠습니다.
* 팀 프로젝트에서는 PM 역할을 맡아 전체 일정과 역할을 조율하고, 구성원 간 협업이 원활히 이루어지도록 조정했습니다.
* 실제 LG HelloVision VOD 시청 데이터를 활용하여 맞춤형 콘텐츠 추천 플랫폼을 구현했습니다.
* React/Vite 기반 프론트엔드와 Node.js/Express/MySQL 기반 백엔드를 구성했습니다.
* JWT 인증을 통해 사용자별 시청 이력, 장르 통계, 시청 패턴, 맞춤형 추천 결과를 제공하는 구조를 구현했습니다.
* VectorDB와 콘텐츠 임베딩을 활용하여 사용자 데이터가 부족한 Cold Start 문제를 완화했습니다.
* AWS에 서비스를 배포하여 시연 현장에서 참여자들이 직접 접속해 사용할 수 있도록 구성했습니다.
* 이 과정에서 기술적인 학습뿐만 아니라, 사람을 설득하고 책임을 나누며 문제를 함께 해결하는 리더십의 본질을 체험했습니다.

## 대표 프로젝트 요약

### TOPIK 모의시험 및 취약 유형 기반 학습 추천 시스템

Repository: [TOPIK](https://github.com/whan1569/TOPIK)

Node.js, Express, EJS, MySQL을 사용해 TOPIK 모의시험 웹 서비스 프로토타입을 구현했습니다.
사용자가 모의시험을 응시하면 정답/오답을 판별하고, 틀린 문제의 유형을 분석해 자주 틀리는 유형과 유사한 문제를 다시 노출하는 취약 유형 기반 추천 흐름을 구현했습니다.
TOPIK 문제 번호별 고정 출제 유형을 활용해 Cold Start 상황에서도 동작 가능한 규칙 기반 추천 구조를 설계했으며, 해당 아이디어를 긍정적으로 평가받아 관련 포럼에 참석했습니다.

### LG HelloVision VOD 데이터 기반 맞춤형 추천 플랫폼

실제 구현 Repository: [team3_web](https://github.com/whan1569/team3_web)
기획·분석 Repository: [Team3](https://github.com/whan1569/Team3)

LG HelloVision의 실제 VOD 시청 데이터를 활용하여 개인화 콘텐츠 추천 플랫폼을 구현한 팀 프로젝트입니다.
React/Vite 기반 프론트엔드와 Node.js/Express/MySQL 기반 백엔드를 구성하고, JWT 인증을 통해 사용자별 시청 이력, 장르 통계, 시청 패턴, 맞춤형 추천 결과를 제공했습니다.
VectorDB와 콘텐츠 임베딩을 활용해 사용자 데이터가 부족한 Cold Start 문제를 완화했으며, AWS에 서비스를 배포하여 시연 현장에서 참여자들이 직접 접속해 사용할 수 있도록 구현했습니다.

## 결론

다양한 사회 경험을 통해 인간관계, 조직 운영, 문제 해결, 기술 적용에 대한 깊은 이해를 갖추게 되었습니다.
이러한 경험들은 제가 개발자뿐만 아니라 다방면에서 유연하게 대처할 수 있는 사람으로 성장하는 데 큰 도움이 되었습니다.

산학실습에서는 기술 아이디어를 실제 프로토타입으로 연결하는 경험을 했고, LG헬로비전 DX DATA 교육에서는 실제 기업 데이터를 활용한 추천 시스템을 구현하며 데이터 기반 서비스의 전체 흐름을 경험했습니다.
가게 운영과 창업 경험은 문제 해결을 넘어 가치를 설계하고 최적의 타이밍을 포착하는 감각을 길러주었습니다.

앞으로도 사회적 경험을 바탕으로 더 나은 방향을 모색하며, 기술적 능력과 인간적 소통을 동시에 강화해 나가고자 합니다.
