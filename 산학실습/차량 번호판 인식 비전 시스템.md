## 차량 번호판 인식 비전 시스템

### 개요

대학교 산학회사 실습 과정에서 **Android 환경과 OpenCV를 연동한 차량 번호판 인식 기능**을 구현했다.
차량 이미지에서 번호판 영역을 검출하고, 전처리 및 영역 분할 과정을 거쳐 번호판 내 숫자를 인식하는 이미지 처리 파이프라인을 구성했다.

### 사용 기술

* Android
* OpenCV
* YOLO
* Computer Vision
* Image Processing
* Object Detection
* Binarization
* Region Segmentation

### 구현 내용

* Android 환경에서 OpenCV 기반 이미지 처리 로직 연동
* YOLO를 활용한 차량 번호판 또는 관심 영역 검출
* 입력 이미지를 Grayscale로 변환하여 색상 정보 제거
* 노이즈 제거 및 이미지 클렌징 처리
* 이진화를 통해 숫자 영역과 배경 영역을 0/1 값으로 분리
* 번호판 내부의 숫자 후보 영역을 개별 영역으로 분할
* 분할된 영역의 픽셀 패턴과 형태를 기반으로 숫자 판별

### 처리 흐름

```text
차량 이미지 입력
→ YOLO 기반 번호판 영역 검출
→ OpenCV 기반 Grayscale 변환
→ 노이즈 제거 및 클렌징
→ 이진화 처리
→ 숫자 영역 분할
→ 숫자 판별
→ 번호판 인식 결과 출력
```

### 핵심 경험

YOLO 기반 객체 검출과 OpenCV 기반 전통 이미지 처리 기법을 조합하여 번호판 인식 파이프라인을 구성했다.
객체 검출 모델로 관심 영역을 좁힌 뒤, Grayscale 변환, 클렌징, 이진화, 영역 분할을 통해 숫자 인식이 가능한 형태로 데이터를 가공하는 과정을 경험했다.
